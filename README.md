🏀 Player Explorer App

A modern Player Search & Management Web Application built using Vanilla JavaScript, HTML, and CSS, powered by a real-world sports API.

⸻

🚀 Live Features

🔐 Authentication
	•	User Signup & Login
	•	Session persistence using localStorage
	•	Logout functionality
	•	Protected routes (redirect if not logged in)

⸻

📊 Player Explorer
	•	Fetch real-time player data from API
	•	Display players in a responsive card layout
	•	Shows:
	•	First Name
	•	Last Name
	•	Team
	•	Position

⸻

🔍 Search
	•	Search players by name
	•	Debounced input for optimized performance

⸻

↕️ Sorting
	•	Sort players:
	•	A → Z
	•	Z → A

⸻

🎯 Filtering
	•	Filter players by team using dropdown

⸻

📄 Pagination
	•	Navigate through players with Next / Previous buttons
	•	Handles API pagination

⸻

⭐ Favorites System
	•	Add/remove players to favorites
	•	Stored per user in localStorage
	•	Dedicated favorites section

⸻

⚡ User Experience
	•	Loading spinner while fetching data
	•	Error handling UI
	•	“No results found” state
	•	Fully responsive design (mobile + desktop)

⸻

🛠️ Tech Stack
	•	HTML5
	•	CSS3
	•	Vanilla JavaScript (ES6+)
	•	Fetch API
	•	LocalStorage
