🏀 Statify

📌 Overview

Statify is a web application that allows users to explore NBA player data in an interactive and user-friendly way. It fetches real-time data from a public API and displays it using a clean UI.

⸻

🎯 Purpose
	•	Explore NBA players easily
	•	Practice working with public APIs
	•	Demonstrate JavaScript Higher-Order Functions
	•	Build a responsive UI

⸻

🔗 API Used
	•	balldontlie API
	•	Base URL: https://api.balldontlie.io/v1/

⸻

✨ Features
	•	🔍 Search players by name
	•	🎯 Filter players by conference (East / West)
	•	🔃 Sort players alphabetically
	•	🧑‍🤝‍🧑 Display player cards
	•	📄 Pagination (optional)

⸻

🛠️ Tech Stack
	•	HTML
	•	Tailwind CSS
	•	JavaScript (ES6)
	•	Fetch API

⸻

🧠 Concepts Used
	•	API Integration (fetch)
	•	Higher-Order Functions:
	•	map()
	•	filter()
	•	sort()
	•	DOM Manipulation

⸻

⚙️ How to Run
	1.	Download or clone the repo
	2.	Open index.html in browser
